# AutoDNSServ
DNS 서브 도메인 자동 제작 프로젝트

## 제작 이유
동아리원들이 프로젝트 제작 후 배포시마다 도메인을 새로 구입해야하는 상황이 발생한다.

때문에 미리 구입한 도메인을 기반으로, 

서브 도메인을 웹페이지에서 간단하게 생성해 연동할 수 있도록 함을 목표로 한다.
